# EMA 20/60 + 天门地门 + PA计数 复合指标 (V2.2)

基于 **Pine Script V6** 开发的高阶复合交易系统。本指标深度融合了**双均线趋势带**、**RSI 极值结构支撑/压力 (天门/地门)**、**微观 N字形突破** 以及 **Price Action 顺势回调计数**。

核心设计理念在于：**通过大周期定方向，结构线定边界，微观形态与计数找高胜率入场点**，有效过滤震荡市的噪音信号。

---

## 🎯 核心功能与模块解析

### 模块一：双 EMA 趋势基准系统 (Trend Baseline)
* **动态趋势色带**：默认采用 EMA 20 与 EMA 60。当 EMA 20 > EMA 60 时填充绿色多头背景；反之填充红色空头背景。
* **交叉信号提示**：自动在图表下方/上方标记金叉（绿色向上三角）与死叉（红色向下三角）位置。
* **高阶平滑处理**：保留了原官方模板的多样化选择，支持叠加 SMA、布林带、SMMA (RMA) 等平滑辅助线。

### 模块二：RSI 天门/地门结构 (Support & Resistance)
基于 RSI(3) 的极端超买/超卖背离逻辑，自动寻找关键的转折点，并绘制长度为 60 根 K 线的支撑/压力射线。
* **天门 (红色射线)**：RSI 高位背离 + 阳线确认。代表上方存在强阻力。
* **地门 (绿色射线)**：RSI 低位背离 + 阴线确认。代表下方存在强支撑。

### 模块三：K线 N字形突破染色 (K-Bar N-Pattern)
识别趋势中的微观接力形态（即“涨-跌-涨”或“跌-涨-跌”），配合门线实现强弱过滤。
* **多头 N字突破 (染红色)**：
   * **形态**：阳线 -> 阴线（回调不破前低） -> 阳线（突破前高）。
   * **门线过滤**：仅当价格处于“天门”水平线之上（或未生成天门）时触发，代表突破顶部压力，多头极强。
* **空头 N字突破 (染绿色)**：
   * **形态**：阴线 -> 阳线（反弹不破前高） -> 阴线（跌破前低）。
   * **门线过滤**：仅当价格处于“地门”水平线之下时触发，代表跌破底层支撑，空头极强。

### 模块四：Price Action 顺势回调计数 (H1/H2 & L1/L2)
源自 Al Brooks 的价格行为理论，量化趋势中的“推进”与“回调”次数。结合了严格的**趋势过滤**与**极简显示**策略。
* **多头计数 (H1, H2)**：
   * 仅在 **绿色 EMA 趋势带** 中生效。
   * 经历回调后再次向上发力时触发。若回调跌破前次低点，上升结构破坏，重置为 H1。
   * **上限控制**：最多显示至 H2，过滤震荡市的无意义信号。
* **空头计数 (L1, L2)**：
   * 仅在 **红色 EMA 趋势带** 中生效。
   * 经历反弹后再次向下发力时触发。若反弹突破前次高点，下降结构破坏，重置为 L1。
   * **上限控制**：最多显示至 L2。

## 💡 交易实战应用指南

本指标最强大的地方在于多模块的**共振**。一个高胜率的顺势做多交易模型如下：
1. **看趋势**：背景为绿色 EMA 色带（多头确立）。
2. **看结构**：价格未受到“天门”射线的压制，或已强势突破天门。
3. **找入场**：
   * **激进试仓**：出现 **H1** 信号或首次红色 N字染色时。
   * **稳健入场 (二次确认)**：出现 **H2** 信号。此时意味着多头在经历了两次回调测试后依然守住了阵地，胜率极高。

## ⚙️ 安装与使用说明

1. 复制 `.pinescript` 源码文件中的所有内容。
2. 打开 TradingView，在底部面板点击 **Pine Editor (Pine 编译器)**。
3. 将代码粘贴至编辑器中，点击 **Save (保存)**。
4. 点击 **Add to Chart (添加到图表)** 即可使用。
5. *提示：可在指标的“设置”面板中，独立开启或关闭染色、射线及 H/L 计数功能。*

# EMA 20/60 + 天门地门 + PA计数 复合指标 (V2.2)

基于 **Pine Script V6** 开发的高阶复合交易系统。本指标深度融合了**双均线趋势带**、**RSI 极值结构支撑/压力 (天门/地门)**、**微观 N字形突破**、**基础 Price Action 计数** 以及源自 Ninetrans 体系的 **Price Action A2/A22 高阶均线回调模型**。

## 🎯 核心功能与模块解析

### 模块一：双 EMA 趋势基准系统
* **动态趋势色带**：默认采用 EMA 20 与 EMA 60。EMA 20 > EMA 60 时填充绿色（多头）；反之填充红色（空头）。
* **交叉信号**：自动标记金叉（绿色向上三角）与死叉（红色向下三角）。

### 模块二：RSI 天门/地门结构过滤
基于 RSI(3) 的极端超买/超卖背离逻辑，自动绘制长达 60 根 K 线的支撑/压力射线。
* **天门 (红色射线)**：代表顶部强阻力。
* **地门 (绿色射线)**：代表底部强支撑。

### 模块三：K线 N字形突破染色
* **多头 N字 (染红色)**：阳线 -> 阴线 -> 阳线。仅在价格位于“天门”上方时生效。
* **空头 N字 (染绿色)**：阴线 -> 阳线 -> 阴线。仅在价格位于“地门”下方时生效。

### 模块四：基础 Price Action 计数 (SECTION 5)
* 基础的趋势波段推移计数器，严格标注顺势行情中的 H1/H2 与 L1/L2，最大显示数值为 2。适合喜欢极简 Price Action 理论的交易者。

### 模块五：Ninetrans A2 / A22 高阶回调系统 (SECTION 6)
该模块独立于基础计数，引入了空间概念，量化趋势中**靠近 EMA 均线**的有效回调与接力。
* **A2 信号**：当在 EMA 20 均线附近发生第二次回调逆转失败（H2/L2）时，标记为 **A2(H2)** 或 **A2(L2)**。如果距离均线太远（超出设定的 ATR 倍数），则不属于优质 A2。
* **A22 信号**：为 A2 准备的容错入场机会。如果 A2 形态不佳，后续的第三次顺势尝试（H3/L3）将标记为 **A22(H3)** 或 **A22(L3)**。
