# Pine Script V6 交易指标模版

这是一个基于 **Pine Script V6** 开发的增强型均线指标。

### 核心功能
* **双 EMA 系统**：默认 EMA 20 与 EMA 60。
* **趋势色带**：自动填充多头（绿）与空头（红）背景。
* **交叉信号**：自动标记金叉与死叉位置。
* **平滑处理**：保留了原官方模板的多种平滑 MA 与布林带选项。

### 如何使用
1. 复制 `.pinescript` 文件中的代码。
2. 在 TradingView 的 Pine Editor 中粘贴。
3. 点击 "Add to Chart" 即可。

## V2 版本更新说明 (2026-04-05)

在 V1 均线系统的基础上，正式集成了“天门/地门”高阶形态识别逻辑与 K 线染色功能。

### 核心功能增强：
1. **天门/地门射线**：
    - 基于 RSI(3) 的极端超买/超卖背离逻辑。
    - 自动在图表绘制长度为 60 根 K 线的压力/支撑射线。
    - **天门**：RSI 高位背离 + 阳线确认（红色射线）。
    - **地门**：RSI 低位背离 + 阴线确认（绿色射线）。

2. **K 线组合动态染色 (无视门限制)**：
    - **多头增强 (阳-阴-阳)**：检测到特定价格突破组合时，连续 3 根 K 线染为红色。
    - **空头增强 (阴-阳-阴)**：检测到特定价格跌破组合时，连续 3 根 K 线染为绿色。

3. **双系统独立控制**：
    - 专门的“显示设置”分组，支持独立开启/关闭射线或染色，不影响原有的 EMA 20/60 趋势线。

### 技术细节：
- 采用 **Pine Script V6** 标准编写。
- 优化了循环检测算法，确保在高数据量下依然保持渲染流畅。
- 修复了 switch 语句在多条件匹配时的语法兼容性问题。
